Configure Namecheap to work with Heroku custom domains

10 October 2012

Configuring the records of Namecheap-hosted domains to point to your Heroku application requires just a few steps:

On Heroku

  1. From the Heroku dashboard select myapp and then its settings page
  2. Add mydomain.com and www.mydomain.com to the Domains section

On Namecheap

  1. From the Manage Domains dashboard, select mydomain.com
  2. Select All Host Records from the navigation bar on the left
  3. Create three @ records pointing, respectively, to the following IP addresses: 75.101.163.44, 174.129.212.2, 75.101.145.87
  4. Select A (Address) under RECORD TYPE for each of the @ records
  5. Create a www record pointing to proxy.heroku.com
  6. Select CNAME (Alias) under RECORD TYPE for the www record

Wait a few seconds and everything should work!
